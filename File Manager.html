概要

ファイル、フォルダの確認や移動。 ほとんどの場合スマホに内臓されています。
「Dropbox」「Google Drive」「OneDrive」「Box」などのクラウドサービスやPCとの連携ができるアプリもあります

おすすめアプリ

Cx File Explorer


「Cx File Explorer」インストール　googlePlay
WindowsからAndroid

Windows、androidを同じWi-Fi接続にしておきます

Windows設定
  1. フォルダを選択し右クリック
  2. 「プロパティ」
  3. 「共有」タブをクリック
  4. 「詳細な共有」をクリック
  5. 「このフォルダーを共有する」にチェックを入れる。
  6. OKをクリック
android設定
  1. 「Cx File Explorer」アプリ
  2. 「ネットワーク」タブ
  3. 「新規ロケーション」
  4. 「リモート」を選択
  5. ［ローカルネットワーク］
  6. 先ほど設定したパソコンを選択
  7. ”WindowsPCの”ユーザー名とパスワードを入力
クラウドサービス

「Dropbox」「Google Drive」「OneDrive」「Box」に対応しています

  1. 「Cx File Explorer」アプリ
  2. 「ネットワーク」タブ
  3. 「新規ロケーション」
  4. 「クラウド」を選択し使用するクラウドサービスをクリックします

WindowsからAndroid

Windows、androidを同じWi-Fi接続にしておきます

Windows設定
  1. フォルダを選択し右クリック
  2. 「プロパティ」
  3. 「共有」タブをクリック
  4. 「詳細な共有」をクリック
  5. 「このフォルダーを共有する」にチェックを入れる。
  6. OKをクリック
android設定
  1. 「リモート」
  2. 「リモートロケーションを追加する」をクリック
  3. 「ローカルネットワーク」を選択
  4. 先ほど設定したパソコンを選択
  5. ”WindowsPC”ユーザー名とパスワードを入力
クラウドサービス

「Dropbox」「Google Drive」「OneDrive」「Box」などに対応しています

  1. 「ファイルマネージャー」
  2. 左下「クラウド」を選択
  3. 「クラウドロケーションを追加する」をクリック
  4. 使用するクラウドサービスをクリックします
