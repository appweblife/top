概要

Windows用メモリ解放のフリーソフトになります。 Reduce Memory

ダウンロード

Reduce Memoryのサイトからダウンロード後インストール不要で使用できます。

使い方

  1. ダウンロード後、圧縮ファイルを解凍します。
    Lhaplus(ラプラス)などフリーソフトで解凍できます。
  2. 「ReduceMemory_x64.exe」というファイルをクリック
  3. 「Reduce Memory」が立ち上がり起動します。起動中は自動的にメモリを解放します。 （初期設定では15秒ごと）
    右上Optimizeをクリックすると、その度ごとにメモリ解放をします。
Reduce Memoryの見方 Total MemoryPCの総メモリ容量
Used Memory使用中のメモリ容量
Free Memory使用可能なメモリ容量

詳細設定

Optionsをクリック

Optimize memory every □seconds メモリ解放の間隔を調整できます。
