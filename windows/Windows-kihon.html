エクスプーラー

Windows上のファイルやフォルダを管理するシステム

Program FilesとProgram Files (x86)

cドライブにあるフォルダ。インストールしたプログラム、ソフトが保存されます

  • [Program Files]　64bitのソフトウェアが保存されます
  • [Program Files (x86)]　32bitのソフトウェアが保存されます
GPU

Graphics Processing Unit。「画像処理装置」のこと。グラフィック専門の処理


グラフィックボードGPU等が搭載されている。”NVIDIA社のGeForce”と”AMD社のRADEON”が有名 コンポーネント ソフトやアプリなどを構成する部品、ファイルなど

    表示の設定

    背景、ロック画面などの設定

    1.設定

    2.個人用設定　
    背景

    自分の好きな背景にカスタマイズできます

    1. 「設定」
    2. 「個人用設定」
    3. 左側から「背景」をクリック
    4. 「画像」「単色」「スライドショー」から選択出来ます
    5. 「参照」をクリックするとPCのフォルダ、ファイルの中から好きな画像を選択できます
    ロック画面

    自分の好きなロック画面にカスタマイズできます

    1. 「設定」
    2. 「個人用設定」
    3. 左側から「ロック画面」をクリック
    4. 「画像」「Windowsスポットライト」「スライドショー」から選択出来ます
      ※「Windowsスポットライト」Microsoft提供で世界中の画像を日替わりで表示する機能
    5. 「参照」をクリックするとPCのフォルダ、ファイルの中から好きな画像を選択できます
    色

    デフォルトのブルーから変えることが出来ます

    1. 「設定」
    2. 「個人用設定」
    3. 左側から「色」をクリック
    4. 「Windowsの色」から設定したい色を選択
    5. デフォルトのブルーから選択した色に変わりました

    スタートの表示

    スタートにアプリやフォルダ表示の設定

    1. 設定
    2. 個人用設定　
    3. スタート
      • 「最近追加したアプリを表示」スタートの上部に”最近追加したアプリ”を表示
      • 「よく使うアプリを表示する」 スタート上部によく”使うアプリ”を表示
      • 「ときどきスタートメニューにおすすめのアプリを表示する」スタートに”マイクロソフトからおすすめのアプリ”が表示されます
    4. スタートに表示するフォルダーを選ぶ
      スタートの左側に [ダウンロード]、[ピクチャ]、[ビデオ]、[エクスプローラー]、[設定]
      などフォルダのアイコンを表示させることが出来ます
    5. 先ほど選択した機能フォルダが表示されます

    タスクバー調整

    タスクバーで何もないところで右クリックするといろいろ設定ができます
    タスクトレイ

    PC画面右下、タスクバー右端に表示される。正式には「通知領域」と言います。
    インターネットやwifiの接続、音量の表示のアイコンなどがあります

    △をクリックすると隠れているアイコンが表示されます
    • [システムアイコン] 音量などのWindowsのシステムのアイコン
    • [アプリケーションアイコン] インストールしたセキュリティソフトなどのアイコン
    表示/非表示 通知領域に表示/非表示することができます
    1. タスクバーで何もないところを右クリックします
    2. 「タスクバーの設定」をクリックし「通知領域」までスクロールします
      • [システムアイコンのオンオフ切り替え]
      • [タスクバーに表示するアイコンを選択します]
    Cortana
    • 場所
    • 表示/非表示
    検索ボックス
    • 検索ボックス

    タスクビュー

    アプリを画面上に表示し切り替えることができます。


    ショートカット［Windows］＋［Tab］キー

    Windows10搭載アプリ

    People

    アドレス帳になります。メールアドレスや住所、電話番号を登録できます。

    Microsoft Solitaire Collection

    トランプゲームになります。

    フィードバック Hub

    マイクロソフトにWindows10の問題点などを報告するアプリ

    マップ

    世界地図、目的地の表示や経路、所要時間などを検索できます。 航空写真表示にもできます。
    1. 左下スタートボタンをクリック
    2. マップをクリック

    OneDrive

    マイクロソフトのオンラインストレージ。5Gまでは無料で利用できる。
    Microsoftアカウントが必要。Mac、iPhoneでも利用可。

    Microsoftアカウント

    作成にはメールアドレスのほか、SMSが受信できる電話番号が必要。
