概要

Windows10標準アプリ。　

「画像」、「動画」のトリミングや回転、傾き調整、フィルター、スライドショー、エフェクト、テキスト入力、再生速度設定等の基礎的な編集が出来ます。
また、iPhoneやAndroid、デジカメの画像をWindowsに取り込むこともできます。

画像編集

画像表示

初期設定ではPCの「ピクチャ」フォルダの画像が表示されます。 基本的に「ピクチャ」フォルダの画像以外の画像は初期設定では表示されません。
画像が保存されているフォルダを新たに追加する必要があります。 スマートフォンやデジカメを接続し取り込む 表示されない画像の追加

画像編集

トリミング　編集と作成→編集→クロップと回転

描画 編集と作成→描画　 ペンで線が描け、消しゴムで消せます。ペンを長押しすると色選択できます。

動画編集

それぞれ幾つか方法があります

トリミング

トリミング
  1. トリミングする動画を右クリックして「プログラムから開く」→「フォト」を選択
  2. 右上の「編集と作成」
  3. トリミング
    • "左右の白丸をドラッグ"し開始位置と終了位置をそれぞれ設定。
    • "白丸を長押し"すると切り取り部分の合計時間が表示されます
    • "青丸をドラッグ"するとプレビュー画面になります
  5. 右上「名前を付けて保存」もしくは「コピーの保存」
トリミング（ビデオエディターから）
  1. 「ビデオエディター」
  2. 「新しいビデオプロジェクト」
  3. ビデオの名前を入力、スキップして後でも可能
  4. 動画を左上「プロジェクト ライブラリ」に動画をドラッグドロップ。または「追加」から
  5. 動画を「ストーリーボードに配置」をクリック。または下部の「ストーリーボード」にドラッグ&ドロップ
  6. 「ストーリーボード」の「トリミング」をクリック。
  7. トリミングの方法
    • 左右の青のマークをドラッグしてトリミング開始、終了位置を設定
    • 白丸をドラッグしてトリミング箇所を確認
    • バーの真ん中の青の部分にカーソルを合わせるとトリミング時間を表示
  8. 右下「ビデオの完了」をクリック。
  9. 「エクスポート」をクリック。
  10. ファイル名と保存先を決定

    11. 分割

    分割 複数回分割して動画の中の部分を削除する場合などに使用します
    1. 動画を右クリックして「プログラムから開く」→「フォト」を選択
    2. 右上の「編集と作成」から「テキスト入りビデオの作成」
    3. 動画を下部の「ストーリーボード」にドラッグ&ドロップ。または動画を「ストーリーボードに配置」をクリック。
    4. 分割したい動画を選択し「ストーリーボード」にある分割をクリック
    5. 白丸（シーク）を動かし分割する位置を決定し完了をクリック

    6. 必要に応じて複数回分割し、必要ない動画を削除
    7. 完成したら右上「ビデオの完了」をクリック

    動画の結合

    1. 左下「スタート」
    2. 「ビデオエディター」をクリック
    3. ビデオの名前を入力、スキップして後でも可能
    4. 左上の「プロジェクト ライブラリ」に結合したい動画をドラッグドロップ、または「追加」から
    5. 下部の「ストーリーボード」に結合する動画をドラッグ&ドロップ
    6. 動画の順番を変える場合は「ストーリーボード」でドラッグし動画を並び替えます
    7. 右上の再生をクリック
    8. 「ビデオの完了」をクリックします。
    9. エクスポートします。※時間がかかることがあります。

    テキスト

    テキストの表示位置、スタイル、表示時間等を設定できます。

    1. 動画を右クリックして「プログラムから開く」 →「フォト」を選択
    2. 「編集と作成」から「テキスト入りビデオの作成」をクリック
    3. 動画を下部の「ストーリーボード」にドラッグ&ドロップ。または動画を「ストーリーボードに配置」をクリック。
    4. ストーリーボード内の「テキスト」をクリック
    5. テキストを入力し、「テキストスタイル」と「レイアウト」から表示位置を決めます。
    6. テキストが表示され、白丸シークをドラッグするとプレビュー画面を確認できます。青のバーをドラッグして表示時間を設定します
    7. 完成したら右上「ビデオの完了」をクリック

    タイトルカード

    表示やエンディングなどになる部分の作成

    1. 動画を右クリックして「プログラムから開く」→「フォト」を選択
    2. 右上の「編集と作成」から「テキスト入りビデオの作成」
    3. 動画を下部の「ストーリーボード」にドラッグ&ドロップ。または動画を「ストーリーボードに配置」をクリック。
    4. ストーリーボード内の 「タイトルカードの追加」
    5. 「タイトルカード」の詳細設定
      • 「表示時間」
        ストーリーボード内の 「期間」をクリックし表示時間を設定できます
      • 「背景色」
        ストーリーボード内「背景」をクリックし背景色を設定します
      • 「テキスト」
        ストーリーボード内「テキスト」をクリックし、タイトルカードにテキストを上下左右中などの位置や太字などのスタイルや表示時間を設定できます。

        • テキストの入力、スタイル、位置
        • テキストの表示時間
    6. ストーリーボード内でドラッグしてタイトルカードの順番を変えることもできます
    7. 完成したら右上「ビデオの完了」をクリック

    描画

    手書き
    1. 動画を右クリックして「プログラムから開く」 →「フォト」を選択
    2. 右上の「編集と作成」
    3. 右上「描画」
    4. 「ボールペン」、「鉛筆」等３種類から選択します。
    5. 線の太さや色を選択
    6. 動画の表示開始位置を決めます。
    7. 消しゴムで消せます。ドラッグして消すのではなく消したい箇所をクリックします。
    8. 右上「コピーを保存」

    スローモーション

    「スローモーション」の速さと開始、終了位置やトリミングを設定できます


    ＊「ビデオエディター」からは出来ません。
    ＊一度に複数のスローモーションを設定することはできません
    1. 動画を右クリックして「プログラムから開く」→「フォト」を選択
    2. 右上の「編集と作成」
    3. 「スローモーションの追加」
    4. 上部の亀と人のアイコンで速度の調整をします。
      目盛りごとに「元の速度」、「さらにスロー」、「スパースロー」の3つに調整できます
      5. ● 「スーパースロー」の場合

      ● 「さらにスロー」の場合
      • 画面の青丸でスローモーションの開始位置と終了位置をそれぞれ指定。※スーパースローの場合、部分でのみ可能
      • 下の〇をドラッグし再生位置を確認出来ます。
      • 左右の白丸をドラッグして動画開始、終了の切り取り、トリミングが出来ます。
    5. 完成したら画面右上にある「コピーを保存」をクリック
    6. 保存先の名前と場所を決めて保存

    写真保存

    1. 動画を右クリックして「プログラムから開く」→「フォト」を選択
    2. 右上の「編集と作成」
    3. 「写真の保存」をクリック
    4. ●バーの部分をクリックまたはドラッグして切り取り位置を決定します。もしくは下の左右の← →をクリックするとコマ送りになります。

      ●バーの部分を長押しすると現在の再生時間が表示されます。
    5. 右上「写真の保存」をクリック

    回転

    1. 回転させる動画を右クリックして「プログラムから開く」→「フォト」を選択
    2. 「テキスト入りビデオの作成」をクリック
    3. ビデオの名前を入力※スキップして後でも可能
    4. 右下"回転のアイコン"をクリック
    5. 右上「ビデオの完了」をクリック
    6. 「エクスポート」
    7. ファイル名と保存先を決定

    3D効果

    雪やハートなど、さまざまなエフェクトを追加できます。表示場所や大きさ、表示時間等も設定できます。

    1. 動画を右クリックして「プログラムから開く」→「フォト」を選択
    2. 右上の「編集と作成」
    3. 「3D効果の追加」をクリック
    4. 適用させたいエフェクトを右側の「効果」から選択します。複数適用させることも可能です。
      「部分的に入るエフェクト」と「画面全体に入るエフェクト」があります
      • 右上の×をクリックするとエフェクトを取り消すことができます。
      • 効果をドラッグすると位置を変えることができます。また、周囲の四角をドラッグすると大きさを変えることができます。
      • 下にあるバーでエフェクトの開始時間と終了時間を決めます。 白丸をドラッグすることによってプレビューも確認できます。
    6. エフェクトを複数選択した場合
    7. エフェクトの設定が終わったら右下「コピーを保存」（ビデオエディターの場合「完了」）をクリックします

    フィルター

    セピア調など、13種類のいろいろな色調のフィルターをかけることができます。
    1. 「フィルター」をクリック
    2. 右側から好みのフィルターを選択し「完了」をクリック
    3. 「エクスポート」をクリック。
    4. ファイル名と保存先を決定

    再生速度

    1. 動画を右クリックして「プログラムから開く」→「フォト」を選択
    2. 「テキスト入りビデオの作成」をクリック
    3. ビデオの名前を入力※スキップして後でも可能
    4. 「ストーリーボード」にある右下「速度」をクリック。
    5. 「クリップの速度を設定する」から速度を設定。
    6. 右上「ビデオの完了」をクリック。
    7. 「エクスポート」をクリック。
    8. ファイル名と保存先を決定

    スライドショー作成

    1. 左下「スタート」
    2. 「ビデオエディター」
    3. 「新しいビデオプロジェクト」
    4. ビデオの名前を入力、スキップして後でも可能
    5. 左側「追加」をクリックしフォルダからスライド用の画像を選択し「開く」をクリック。
      または「プロジェクト ライブラリ」へスライド用の画像をドラッグドロップ
    6. 画像を選択し「ストーリーボードに配置」をクリック。または「ストーリーボード」にクリックドラッグ
    7. 画像を選択し「期間」から各画像の表示時間を調整
    8. 画像の並び替えは「ストーリーボード」上の画像をドラッグ
    9. 右上「ビデオの完了」をクリック
    10. 「エクスポート」をクリック。
    11. ファイル名と保存先を決定

    プロジェクト

    新規編集作成と自動保存された動画を再編集できます。

    新規編集作成
    1. 左下「スタート」
    2. 「ビデオエディター」
    3. 「新しいビデオプロジェクト」
    4. ビデオの名前を入力、スキップして後でも可能
    再編集 「マイビデオプロジェクト」に保存され、再編集が可能です
    1. 左下「スタート」
    2. 「ビデオエディター」
    3. 「マイビデオプロジェクト」から再編集したいビデオ（動画）を選択します
    カバー写真 表示される「カバー写真」は自動的に作成されます 「マイビデオプロジェクト」に保存され、再編集が可能です
    1. 左下「スタート」
    2. 「ビデオエディター」
    3. 「マイビデオプロジェクト」からカバー写真を変更したい動画を選択します
    4. 左上「プロジェクトライブラリ」の「追加」をクリックします
    5. ※PC内にある画像を使用の場合は「このPCから」をクリック
    6. 画像を選択追加後、左上「ビデオエディター」をクリックします
    7. 「マイビデオプロジェクト」から画像を追加したビデオを選択し右クリック、「カバー写真の変更」をクリックします
    8. 変更する画像を選び、右上「完了」をクリック
    ビデオタイトル（動画名）の変更 「マイビデオプロジェクト」に保存され、再編集が可能です
    1. 左下「スタート」
    2. 「ビデオエディター」
    3. 「マイビデオプロジェクト」から名前を変更したいプロジェクト（動画）を選択
    4. 左上”鉛筆のマーク”をクリック
    5. 任意の名前を入力しOKをクリック
    6. 入力後、左上の「ビデオエディター」をクリックします
