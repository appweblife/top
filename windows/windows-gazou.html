ペイント

Windowsに搭載されていて 写真や画像に色やフォントなどを自由に設定して、文字入れできます。
右下スタートボタン→windowsアクセサリ→ペイント

  • 画像サイズ変更 　縦横のサイズ。ピクセルまたは%での指定。
  • トリミング　「選択」→「図形の選択」→トリミングする範囲をドラッグ→「トリミング」クリック。
  • 画像に文字入れ　「ツール」の中の「Ａ」マーク
  • 線　「ツール」の中の鉛筆。shiftを押しながら引くと、直線になります。

ペイント3D

ペイントからでも開ける

テキスト　いろいろなフォントがある
図形　3D,2Dのいろいろな図形を挿入できる
トリミング　画像の周囲6つの点を動かし決定
保存　3D,2Dなどで保存できる

リサイズ

右下の『キャンパスのサイズを変更する時に、画像も調整します』を必ずオンにする。
そうするとキャンパスサイズと一緒に画像のサイズも変更される。
また、「縦横比を固定する」にチェックをいれる

幅と高さは、ピクセルかパーセント。

音楽

CDの取り込み、再生

Windows Media Playerを使用します。

  1. 左下「スタートボタン」をクリック
  2. 「Windows アクセサリ」から「Windows Media Player」をクリック
CDの取り込み
  1. CDをPCにセットします。
  2. 曲名が表示されたら取り込みたい曲にチェックを入れます
  3. 上部「CDの取り込み」をクリック

「PC」の 「ミュージック」フォルダに保存されます。
