エクスプーラー

エクスプーラーを開く ［Windows］＋[E]
エクスプーラーの表示タブ ［Alt］＋［V］
エクスプーラーの検索ボックス
［Ctrl］＋［E］または［Ctrl］＋［F］
［End］一番下へ
［Home］一番上へ
[shift]+ [→][←] [↑][↓] または[shift]を押しながらクリック 連続した項目を複数選択
［ctrl］を押しながら選択したい項目をクリック離れた項目を複数選択

web

  • 「タブ複製」　Alt+D+Enter
  • 「Webページの更新」　F5
  • 「履歴を開く」　Ctrl+H
  • 「ダウンロード履歴を開く」　Ctrl+Ｊ
  • 「ブックマークを開く」　Ctrl+B
  • 「チェックをいれる」　スペースキー
  • 「ページのコード表示(IE以外)」　Ctrl+U
  • 「ページの末尾に移動する」　Ctrl+End
  • 「ページの先頭に移動する」 Ctrl+Home
  • 「ページを戻る」〔 Alt 〕＋〔 ← 〕
  • 「ページを進む」 [ Alt ]＋〔 → ]
  • 「タブを切り替える」〔 Ctrl 〕＋〔 Tab 〕
  • 「タブを右に移動」　Ctrl＋Tab
  • 「最初のタブ」　Ctrl＋1
  • 「最後のタブ」　Ctrl＋

excel

  • 下のセルへの移動はEnterキー
  • 右のセルへの移動はtabキー
  • 直前の操作繰り返し　　F4
  • 検索　Ctrl＋F
  • 太字設定、解除　　Ctrl＋B
  • 下線設定、解除　　Ctrl＋4
  • ファイルを開く　　Ctrl＋O
  • 印刷プレビュー　　Ctrl＋F2
  • シートの切り替え　　Ctrl＋pgup or pgdn (のーとPCはFnをプラス）

word

  • 印刷プレビュー　Ctrl＋F2
